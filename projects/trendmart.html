Project Highlights

Trend Mart combines modern UI, real-time performance, and secure backend infrastructure to simulate a complete online store environment.

JWT Authentication

Implemented secure login and signup using JSON Web Tokens (JWT) and bcrypt for password encryption. Users and admins have separate access privileges, ensuring data integrity and restricted controls.

Product Management

Admin users can create, update, and delete products, along with managing categories, pricing, images, and stock. CSV data was seeded automatically into MongoDB using Node.js for scalability testing.

Cart & Wishlist

Dynamic cart with real-time quantity updates, total calculations, and persistent data storage using localStorage. Includes wishlist functionality to enhance the user shopping experience.

Development Journey

Trend Mart was developed as a real-world simulation of a professional e-commerce platform. The development process included backend API design, authentication setup, and dynamic UI creation.

The application features full CRUD operations, live search, product filters, and payment options. It integrates UPI & Visa payment methods and generates order receipts dynamically using PDFKit. The admin dashboard offers real-time control over users, products, and orders with secure access management.

Dark mode support, toast notifications, and responsive design were implemented to enhance user experience. Finally, the project was deployed using a dual-service structure: the backend via Render and the frontend via Vercel.
